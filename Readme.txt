Overview
This project is a basic calculator application developed using Kotlin. The calculator allows users to perform basic arithmetic operations, manage input through backspace, clear inputs with an AC button, and compute results using an equals sign.

Features
Number Input: Input numbers using digit buttons (0-9).
Basic Operators: Perform arithmetic operations with addition (+), subtraction (-), multiplication (*), and division (/).
Backspace: Remove the last entered digit or operator.
AC (All Clear): Clear the entire input.
Equals (=): Compute and display the result of the arithmetic expression.
Installation
To run this application, ensure you have the following prerequisites:

Kotlin 1.4 or above
An IDE such as IntelliJ IDEA or Android Studio

Usage
Input Numbers: Click on the number buttons (0-9) to input numbers.
Perform Operations: Click on any of the operator buttons (+, -, *, /) to perform arithmetic operations.
Backspace: Click the backspace button to remove the last digit or operator entered.
Clear Input: Click the AC button to clear all input.
Compute Result: Click the equals (=) button to calculate and display the result of the arithmetic expression.
Code Structure
MainActivity.kt: Contains the main activity code for the calculator.
activity_main.xml: Defines the layout of the calculator interface.
